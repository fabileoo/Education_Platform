# Education Platform (ASP.NET Core)

Learning Management System built with ASP.NET Core.

## Tech Stack
- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- REST API
- JWT Authentication
- Git

## Features
- User authentication and authorization
- Role-based access (Admin / User)
- CRUD for courses and educational materials
- File upload and content management
- RESTful API with clean architecture

## Responsibilities
- Backend development
- Database design
- API implementation
- Authentication and authorization logic

## How to run
1. Configure connection string in `appsettings.json`
2. Run database migrations
3. `dotnet run`
